## Aplicação Laravel

- Iniciar a aplicação
```
make build
```

- Acompanhar as mensagens no Apahce Kafka

```
make kafka_consumer_01
```

- Produzir eventos para o Apache Kafka
```
make producer
```

- Consumir mensagens na primeira instância
```
make consumer_01
```

- Consumir mensagens na segunda instância
```
make consumer_02
```
